# NoteBook

Notepad app

   Application features:
- 1. Add, edit and delete notes with details
  * Title
  * Text
  * History
  * Grouping
 - 2. Add, edit and delete categories with details
    * Title
 - 3. Pin the notes to the top of the list
 - 4. Search in the title of the notes
